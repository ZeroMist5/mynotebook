变形

变形

CSS3里面加入了变形，可以用来改变元素的形状。

变形属性

格式：transform:一个一个的变形的方法

2d变形的方法

  • translatex()，沿着元素自身的x轴移动元素

    • px，像素。正数往右，负数往左。

    • %，相对于自身的宽度来计算（包括内容区、内边距、边框，不包括margin）。

        <!DOCTYPE html>
  <html>
      <head>
          <style>
              body{
                  margin:0;
              }
              div{
                  margin-left:100px;
                  /* 240 */
                  padding:10px;
                  /* 240px+100 */
                  transform:translatex(100%);
                  width:200px;
                  height:200px;
                  border:10px solid green;
              }
          </style>
      </head>
      <body>
          <div></div>
      </body>
  </html>

  • translatey()，沿着自身的y轴移动元素。

    • px，像素

    • %，相对于自身的高度来计算。

      transform支持多个方法，你可以在transform后面使用多个方法，多个方法之间使用空格进行分隔。

      如果有多个变形方法，一定要确保每个变形方法都正确使用，否则将会整条规则失效。

      我们可以使用transform:translatex和translatey来进行水平、垂直居中，这样的好处就在于不用去计算自身的宽度和高度。

      <!DOCTYPE html>
<html>
  <head>
      <style>
          #f{
              /* 2. 父元素成为子元素的容纳块 */
              position:relative;
              width:500px;
              height:500px;
              border:1px solid black;
          }
          #z{
              /* 1. 子元素设置绝对居中 */
              position: absolute;
              /* 3. 元素向右移动50%（父元素） */
              left:50%;
              /* 4. 元素向左回自身的一半 */
              /* margin-left:-111px; */

              top:50%;
              transform:translatex(-50%) translateY(-50%);
              padding:10px;
              width:200px;
              height:200px;
              border:1px solid red;
          }
      </style>
  </head>
  <body>
      <div id="f">
          <div id="z"></div>
      </div>
  </body>
</html>

  • translate(x,y)，同时沿着x轴及y轴来进行移动。

    第一个值沿着x轴移动的量，第二个值是沿着y轴移动的量。如果省略y那么假定y为0。

  • scalex()，在x轴上进行缩放。

  • scaley()，在y轴上进行缩放。

    他们的值只能是没有单位的数，始终得是正数，可以是小数，值的意思是宽度或高度的倍数。

    经过变形之后的元素可能个比之前大也可能比之前小，不过元素在页面上所占据的空间保持不变。

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              img:hover{
                  transform:scaleX(.5) scaleY(.5);
              }
          </style>
      </head>
      <body>
          <img src="./u=1931042946,2154311841&fm=26&gp=0.jpg" alt=""><br/>
          我瘦了4斤。。。
      </body>
  </html>

  • scale(x,y)，x轴和y轴同时进行缩放。

    如果只写一个值的时候将会将这个值同时应用在x轴和y轴上。

  • skewx()，在水平方向扭曲变形，如果值为正数，元素的左上角和右下角会被拉扯，如果值为负数，元素的右上角和左下角会被拉扯

  • skewy()，在水平方向扭曲变形，如果值为正数，元素的左上角和右下角会被拉扯，如果值为负数，元素的右上角和左下角会被拉扯。

  • rotatez()、rotate()，沿着z轴旋转指定角度。

    正值顺时针，负值逆时针。

    旋转时要注意45deg和405deg的区别，如果不加过渡看不出区别，但是加上过渡之后就会显示出来多转一圈。

    旋转会更改x轴和y轴。

      transform: translatex(100px) rotate(45deg) ;

    上面的代码是先移动，然后再旋转（x和y产生的变化）

      transform:  rotate(45deg) translatex(100px);

    上面代码是先旋转（x和y产生了变化），然后再移动。

  • 移动原点

    默认的时候变形都以元素的绝对中心作为变形的原点，transform-origin可以更改原点的位置。

      <!DOCTYPE html>
  <html>

  <head>
      <style>
          img {
              width: 200px;

          }

          img:hover {
              /* transition: transform 1s;
              transform: translatex(300px) rotate(45deg); */
              transform:rotate(45deg);
              /* 左上角 */
              /* transform-origin:0 0; */
              /* 右上角 */
              transform-origin: 100% 0;
          }
      </style>
  </head>

  <body>
      <img src="./big103015-1.png" alt="">
  </body>

  </html>

动画

过渡：在一段时间内将属性的值变为另外一个值。在一段时间内完成的。

动画也是让属性的值在一段时间内发生变化，但是动画会对这些控制权更大。

定义步骤：

  1. 定义关键帧。
  2. 在元素上使用。

  3. 指定动画的持续时间。

  4. 定义关键帧

    你想定义动画就需要有一个规则，这个规则就需要有个名字，这个规则能够反复使用。

      @keyframes 规则名{
  }

  @keyframes test{
  }

    这个关键帧里面需要有多个关键帧的组成块，每个关键帧的组成块包括动画持续的时间点和你要变化的属性、值

      @keyframes myTest{
      0%{
          property1:value1;
          .....
      }
      100%{
          property1:value2;
          ......
      }
  }

    0%表示的是动画刚开始的时候。

    100%表示动画结束的时候。

    75%表示动画（持续时间）进行到75%的时候

  5. 在元素或伪元素上使用动画

    animation-name:规则名

  6. 设定动画的持续时间

    只是定义了动画和将动画应用到某个元素上不足以展示出动画效果。所以需要让动画持续一定的时间。

    animation-duration:value

    持续时间需要带上单位：s或ms。

    transition-duration:value

    注意：

    1. 可以使用from和to两个关键字分别表示0%和100%。

    2. 如果你没有指定0%或100%的时候，浏览器将要应用动画效果的属性的原始值自动构建成一个0%和100%。

      ```css

      @keyframes mytest {

        /*0%{
            background-color:yellow;
        }*/
        50%{
            background-color:pink;
        }
        /*
        100%{
            background-color: yellow;
        }*/
    }

    ```

3. 关键帧的块时间点的顺序不必按照升序来排列。

    ```css
    5%{
    }
    75%{
    }
    10%{
    }
    ```

4. 一个动画上面可以应用多个动画

    `animation-name:规则1,规则2......`

    一般来说animation-name有多少个值，animation-duration就需要有多少个值。

    ```html
    <!DOCTYPE html>
    <html>

    <head>
        <style>
            #f {
                width: 300px;
                height: 300px;
                border: 1px solid yellow;
                animation-name: mytest,mytest1;
                animation-duration: 10s;
                background-color:yellow;
            }

            @keyframes mytest {
                100%{
                    background-color: pink;
                }
            }
            @keyframes mytest1{
                100%{
                    font-size:100px;
                }
            }
        </style>
    </head>

    <body>
        <div id="f">
            真英俊！
        </div>
    </body>

    </html>
    ```

    1. 如果指定的规则名不存在，不会导致整个动画都无法使用，只是不存在动画将会被忽略。其他的该有效果还是有效果的。
    2. 多个动画中如果有重复的属性，后面的动画覆盖前面的动画中的相同属性。

  • 延迟播放动画

    默认的情况下动画上来就开始执行，没有延迟。

    animation-delay来设置第一次延迟时间。

    如果多次执行动画都要有延迟那么需要用到JS。

  • 动画的迭代次数

    animation-iteration-count

    默认的时候动画只是执行了一次。

    • 1，默认值。
    • 任何的数字。
    • infinite，无穷大。

  • 设置动画的播放方向

    animation-direction

    • nomral，默认值。每次迭代都从0%~100%。
    • reverse，每次从100%到0%。
    • alternate，动画在奇数次(1\3\5\7\9.....)正向播放(0~100%)，在偶数次(2\4\6\8......)反向播放(100%~0)。
    • alternate-reverse，动画在奇数次(1\3\5\7\9.....)反向播放(100%~0%)，在偶数次(2\4\6\8......)正向播放(0~100%)。

  • 改变动画的内部时序

    animation-timing-function

    • ease，慢-》快-》慢
    • linear，线性的、匀速的。
    • ease-in，慢->快
    • ease-out，快-》慢
    • 贝塞尔曲线
