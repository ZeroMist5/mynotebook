边框与外边距

复习

内联盒子模型：

种类：匿名内联盒子、具名内联盒子。 

<p>呵呵<b>!</b>?</p>

组成部分：内容区（放置具体内容的）、上下半行间距（放在内容区上方和下方的空白）

一行一行的内容有可能有多个内联盒子：每一行都会形成一个行框。行框包裹住本行内联盒子。

行高的作用：

  1. 调整行与行之间的（最小，可以大于等于）距离。
  2. 单行文本垂直居中（行高等于高度）。

vertical-align：

  • baseline，默认值。基线对齐。
  • top，内联盒子的顶部与行框的顶部对齐。
  • bottom，内联盒子的顶部与行框的底部对齐。
  • middle，与父元素的x的1/2处对齐。

vertical-align在使用的时候出现的问题：

  1. 一行中，一个高的元素占据了整行高度，vertical-align不会对它有影响（因为没有空间），这个时候还需要对齐所以将会导致父元素的基线产生移动。

  2. 块状元素包含一张图片的时候底部有空白。

    分析原因：<!DOCTYPE html>的时候在每行的前面会有一个幽灵节点。对齐的时候还是baseline对齐所以下面的边距就是空白。

    解决问题：

    1. 图片块状化，块状元素没有基线。
    2. font-size:0;
    3. vertical-align:bottom/top/middle

  3. 行内块状元素对齐的问题

    行内块状元素如果没有内容那么基线就是底部，如果里面有内容则按照最后一行的基线来对齐。

盒子模型

组成部分：

  • 内容区：放置具体内容。
    • 宽度：定义内容区的宽度。
    • 高度：定义内容区的高度。
  • 内边距：用来定义内容区和边框的距离。
  • 边框：边框是内容区或内边距周围的一条或多条线。
  • 外边距：定义元素和元素之间的距离。

内边距

内边距适用于所有的元素。元素默认的时候没有内边距，内边距默认是0。

格式：padding:value

  • px，像素。padding不能设置为负数。

  • %，使用百分比来设置内边距。不管是水平的padding还是垂直的padding都是按照父元素的内容区的宽度来计算。

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              #f{
                  width:500px;
                  height:450px;
                  border:1px solid green;
              }
              #z{
                  /* padding 分左右  250*2+100+2 */
                  /* padding 分上下  250*2+100+2 */
                  padding:50%;
                  width:100px;
                  height:100px;
                  border:1px solid red;
              }
          </style>
      </head>
      <body>
          <div id="f">
              <div id="z"></div>
          </div>
      </body>
  </html>

  • 简写格式：

    • 一个值：padding:top/right/bottom/left

    • 两个值：padding:top/bottom left/right

    • 三个值：padding:top left/right bottom

    • 四个值：padding:top right bottom left

      值的格式为顺时针。

  • 设置单边内边距

    • padding-top
    • padding-right
    • padding-bottom
    • padding-left

  • 行内元素的内边距

    一般都是在块状元素和行内块状元素上设置内边距。

    对于行内元素来说左右内边距可以正常使用，但是上下内边距可以生效没有效果。因为控制行内元素的距离的只能是行高。

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              span{
                  padding:100px;
                  border:1px solid green;
              }
          </style>
      </head>
      <body>
          <span>闫海静真英俊！</span><b>abc</b>
          <div style="border:1px solid red">123123</div>
      </body>
  </html>

边框

边框组成有三个部分：

  • 边框的宽度：border-width：具体看浏览器，一般来说边框宽度是2px或3px的默认值。
  • 边框的样式：border-style：默认为none，边框没有样式将不会显示出来边框。
  • 边框的颜色：border-color：边框颜色，默认的边框颜色为文本的颜色。

边框样式

border-style来设置

  • none，没有样式。默认值。当值为none的时候不仅没有样式也没有宽度。
  • solid，实线。
  • 其他值自行百度。

简写格式：

  • 一个值：border-style:top/right/bottom/left
  • 两个值：border-style:top/bottom left/right
  • 三个值：border-style:top left/right bottom
  • 四个值：border-style:top right bottom left

设置单边边框样式

  • border-top-style
  • border-right-style
  • border-bottom-style
  • border-left-style

边框宽度

边框宽度取值：px像素

简写格式：

  • 一个值：border-width:top/right/bottom/left
  • 两个值：border-width:top/bottom left/right
  • 三个值：border-width:top left/right bottom
  • 四个值：border-width:top right bottom left

设置单边边框宽度

  • border-top-width
  • border-right-width
  • border-bottom-width
  • border-left-width

边框颜色

简写格式：

  • 一个值：border-color:top/right/bottom/left
  • 两个值：border-color:top/bottom left/right
  • 三个值：border-color:top left/right bottom
  • 四个值：border-color:top right bottom left

设置单边边框宽度

  • border-top-color
  • border-right-color
  • border-bottom-color
  • border-left-color

transparent表示透明的意思。

边框样式没有和边框颜色是transparent是不同的。

  • 边框样式没有，也就没有斜切角。
  • 边框有样式但是颜色是透明的，会有斜切角。
<!DOCTYPE html>
<html>

<head>
    <style>
        div {
            width: 300px;
            height: 500px;

            border-style:solid;
            border-width:20px;
            border-color:red yellow green pink;
            /* border-top-color:transparent; */
            border-top-style: none;
        }

    </style>
</head>

<body>
    <div>

    </div>
</body>

</html>

简写格式：

格式：border-top:width style color

格式：border-right

格式：border-bottom

格式：border-left

格式：border;width style color

border:1px solid green

行内元素（内联元素）的边框

行内元素上设置上边框和下边框对行高没有影响。

行内元素设置的左边框和右边框对布局是有影响的。

<!DOCTYPE html>
<html>

<head>
    <style>
        #f{
            display:inline;
            width:200px;
            height:200px;
            border:100px solid green;
        }
        #z{

            width:200px;
            height:200px;
            border:1px solid red;
        }

    </style>
</head>

<body>
    <div id="f">abc</div>
    <div id="z">bcd</div>
</body>

</html>

外边距

外边距使用margin属性，外边距是添加在边框外边界的外侧。

浏览器会给很多元素提供默认样式。其中就包括外边距。

body有8px的外边距。

  • 外边距取值

    • auto，自动的。
    • px，像素。默认的时候是0px。
    • %，百分比相对于父元素的内容区的宽度来计算。

  • 简写格式：

    • margin:top/right/bottom/left（一个值）
    • margin:top/bottom left/right（两个值）
    • margin:top left/right bottom（三个值）
    • margin:top right bottom left（四个值）

  • 单边外边距

    • margin-top：上外边距
    • margin-right：右外边距
    • margin-bottom：下外边距

    • margin-left：左外边距。

      元素的外边距是可以设置为负数的。

外边距折叠

块状元素的上下外边距会产生折叠，折叠的时候会折叠为最大的那个。

  • 情况一：兄弟元素折叠

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              div{
                  width:100px;
                  height:100px;
                  border:1px solid red;
              }
              #one{

                  margin-bottom:10px;
              }
              #two{
                  display:inline-block;
                  margin-top:10px;
              }
          </style>
      </head>
      <body>
          <div id="one"></div>
          <div id="two"></div>
      </body>
  </html>

    产生的原因就是因为margin之间没有阻隔直接接触了。

  • 情况二：父子折叠

    ```html <!DOCTYPE html> 

      <head>
      <style>
          h1{
              margin: 10px;
              border:1px solid green;
          }
          #f{
              /* border:1px solid pink; */
              /* padding:1px; */
              background-color:hotpink;
          }
      </style>
  </head>
  <body>
      <!-- 
          #fmargin（0） -> #fborder（0） -> #fpadding（0） -> h1margin（10） -> h1border -> h1padding ->h1的内容区

         -->
        <div id="f">
            <!-- h1margin + h1border + h1padding + h1内容区高度
            20 + 42 = 62
            -->
            <h1>hello world</h1>
        </div>
    </body>
</html>
```

解决方法：给父元素加上边框或内边距。或者给父元素加上`overflow:hidden`

行内元素的外边距

行内元素上设置上下外边距对行高没有影响。但是左右外边距可以正常使用。

横向auto的计算方式

默认情况下块级框组成部分的横向尺寸始终等于容纳块的宽度。

容纳块：离的最近的块状元素。

容纳块的宽度（width）= 子元素的margin-left + 子元素-border-left + 子元素的padding-left + 子元素width + 子元素的padding-right + 子元素的border-right + 子元素的margin-right

七个属性只有width、margin-left、margin-right可以设置为auto，其他的只能设置具体的值或默认值。

  • 情况一：width、margin-left、margin-right其中两个设置为具体值，一个设置为auto。谁为auto必须去满足容纳块宽度公式。auto你可以理解为一个x

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              #f{
                  width:300px;
                  height:300px;
                  border:1px solid green;
              }

              #z{
                  /* 300-100-10-2 */
                  width:100px;
                  height:100px;
                  margin-left:10px;
                  margin-right:auto;
                  border:1px solid red;
              }
          </style>
      </head>
      <body>
          <div id="f">
              <div id="z"></div>
          </div>
      </body>
  </html>

  • 情况二：width、margin-left、margin-right两个设置为auto

    如果是marign-left和margin-right为auto，这个时候两侧的外边距相等。元素在父元素中间显示。

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              #f{
                  width:300px;
                  height:300px;
                  border:1px solid green;
              }

              #z{
                  /* (300-200-2)/2 */
                  width:200px;
                  height:100px;
                  margin-left:auto;
                  margin-right:auto;
                  border:1px solid red;
              }
          </style>
      </head>
      <body>
          <div id="f">
              <div id="z"></div>
          </div>
      </body>
  </html>

    width和margin-left为auto，margin-right具体值。width和margin-right为auto，margin-left为具体值。哪边margin为auto哪边会被解释为0，然后执行容纳块公式，width填满剩下的值。

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              #f{
                  width:300px;
                  height:300px;
                  border:1px solid green;
              }

              #z{
                  /*300-100-2*/
                  margin-left:auto;/*0*/
                  width:auto;/**/
                  margin-right:100px;
                  height:100px;

                  border:1px solid red;
              }
          </style>
      </head>
      <body>
          <div id="f">
              <div id="z"></div>
          </div>
      </body>
  </html>

  • 情况三：width、margin-left、margin-right三个设置为auto。那么margin-left和margin-right都解释为0，width自动运行容纳块公式。

      <!DOCTYPE html>
  <html>
      <head>
          <style>
              #f{
                  width:300px;
                  height:300px;
                  border:1px solid green;
              }

              #z{
                  /*300-2*/
                  margin-left:auto;/*0*/
                  width:auto;
                  margin-right:auto;/*0*/
                  height:100px;

                  border:1px solid red;
              }
          </style>
      </head>
      <body>
          <div id="f">
              <div id="z"></div>
          </div>
      </body>
  </html>

调整宽度和高度的计算方式

width等于内容区的宽度、height等于内容区的高度。

box-sizing来进行调整，可以用在所有可以设定width和height的元素上。

  • content-box，默认值。内容区的宽度等于width、内容区的高度等于height，内边距和边框是在这个上面增加。（w3c盒子模型的计算方式）

    2000年：3000（吃饭） + 抽烟（20*30） + 喝酒（500） + 买衣服（300） = 4400

  • border-box，宽度、高度等于内容区+内边距+边框。这个时候width和height就是一个总数。（IE盒子模型、怪异盒子模型），按照这种计算padding和border越大相对应的内容区就会越小。

    2022年：3000（吃饭+抽烟+喝酒+买衣服）

<!DOCTYPE html>
<html>
    <head>
        <style>
            #f{
                /* 
                200-20-2 

                 */
                 margin:100px;
                box-sizing: border-box;
                padding:10px;
                width:200px;
                height:200px;
                border:1px solid green;
            }
        </style>
    </head>
    <body>
        <div id="f">

        </div>
    </body>
</html>

背景

背景色

background-color，用来设置背景色。

所有元素的背景色都是透明的（transparent）。

<!DOCTYPE html>
<html>
    <head>
        <style>
            html{
                background-color:lightpink;
            }
            body{
                margin:0;
                background-color:lightblue;
            }
        </style>
    </head>
    <body>

    </body>
</html>

整个页面的层次：body -> html -> 页面画布

最终整体背景颜色由页面画布决定（所有的内容都是在页面画布上渲染出来）

画布颜色决定的机制：有html的颜色就用html的颜色，如果html没有颜色就用body的颜色。如果都没有设置则使用画布本身的颜色（透明）页面最终的颜色由浏览器自己决定，一般浏览器都会将页面画布设置为白色。

background-color还可以使用其他颜色的值。

图片也可以设置背景色，但是背景色会被压在图片的下面。

背景会延伸到边框的外边界（默认的时候会包括内容区、内边距、边框）

<!DOCTYPE html>
<html>
    <head>
        <style>
            /* html{
                background-color:lightpink;
            }
            body{
                margin:0;
                background-color:lightblue;
            } */

            img{
                background-color:gray;
                padding:10px;
                border:10px solid red;
            }
        </style>
    </head>
    <body>
        <!-- <img src="./13560462.png" alt=""> -->
        <img src="./1117.jpg" alt="">
    </body>
</html>

图片的内容区就是图片本身。
