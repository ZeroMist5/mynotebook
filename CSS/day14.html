过渡

过渡是元素从一种样式逐渐改变为另外一种样式。

步骤：

  1. 指定要添加效果的CSS属性
  2. 指定过渡效果持续的时间。

transparent 透明

transition 过渡

transform 变形

translate 移动

定义过渡的属性

transition-property，只有在这个属性里面定义的属性才会以动画形式持续一段时间而不是瞬间变化。

  • none，不过渡任何属性

  • all，过渡所有属性（默认值）

    属性值能够使用数字或能够转换为数字一般都支持过渡。

    display:none不支持过渡。

  • 其他的值可以是以逗号分隔的属性的列表。

设置过渡的持续时间

transition-duration，指定过渡持续的时间，从一个状态过渡到另外一个状态需要经过多少时间。

  • 默认时候过渡时间是0。

  • 值的单位可以是s或ms，必须带单位。

    1s=1000ms

  • 过渡的持续时间也可以是列表。中间以逗号进行分隔。

<!DOCTYPE html>
<html>

<head>
    <style>
        div {
           width:300px;
           height:100px;
           border:1px solid green;
        }
        body:hover div{
            width:800px;
            height:150px;
            border-width:10px;
            transition-property:width,height,border-width;
            transition-duration:1s,2s,1s;
        }
    </style>
</head>

<body>
    <div></div>
</body>

</html>

调整过渡方式

默认的时候过渡样式是慢->快->慢。

transition-timing-function来修改

  • ease，慢-》快-》慢

  • linear，线性的，匀速的。

  • ease-in，慢-》快

  • ease-out，快-》慢

  • 指定三次方的贝塞尔曲线，

      https://cubic-bezier.com/#.17,.67,.83,.67

  1. 调整运动轨迹。
  2. 调整之后运动轨迹的预览、效果。
  3. 系统已经给定的参数的效果
  4. 整个效果的持续时间
  5. 执行查看效果的对比
  6. 选择系统给定参数
  7. 直接复制粘贴参数就好了。

延迟过渡

transition-delay经过多长时间之后开始执行过渡。

transition-delay

  • 默认是0s，过渡立即开始没有延迟。
  • 其他的规则和transition-duraition规则相同。

简写格式

格式：traition:property（all） duration（0s） timing-function（ease） delay（0s）

格式：transition:property1 duration1 timing-function1 delay1,property2 duration2 tminig-function2 delay2

反向过渡

我们一般在动态伪类选择器上触发，hover，只有在鼠标移入时才会触发过渡，移出时马上变为原来的状态。

将所有的过渡放在默认的状态上就可以实现所谓的反向过渡，移入的时候有效果，移的时候也会有相同的效果。

反响过渡中如果有延迟，延迟相同

注意：过渡属于锦上添花，不属于雪中送炭。

菜单：

  1. 二级菜单不能影响下面的东西。
  2. 可以自由的增加或减少二级菜单的数量。
  3. 二级菜单的内容长度，如果比较长需要自动撑开，如果比较短那就用父元素的长度。
