背景与列表

从大到小，从外到内，从布局到装饰。

默认背景会延伸到边框的外边界。

背景图

background-image:url(地址)

如果背景图比元素大那么就只显示出来一部分，如果背景图比元素小那么就开始重复。

可以同时设置背景图和背景色，背景图优先显示。

如果这张图片是网页中要表示、标记出来的内容那么用img标签，如果这张图片是用来修饰用的那么使用背景图。

背景图重复方式

background-repeat

  • repeat，默认值，横向平铺、纵向平铺。
  • repeat-x，横向平铺。
  • repeat-y，纵向平铺。
  • no-repeat，不重复、不平铺。

背景定位

background-position

  • 关键字

    • left
    • right
    • top
    • bottom

    • center

      关键字可以使用两个值，第一个指横向的位置，第二指向纵向的位置。

      如果同时指定了left、right或top、bottom那么将会导致语法错误（属性不合法）。

      如果只设置一个值，那么另外一个值默认为center。

  • px，像素

    默认的时候背景图是在元素内边距的外边界的0，0（左上角）的位置来定位

    第一个值为横向偏移量，第二个值是纵向偏移量。

    背景定位第一个值正数往右走，负数往左走。背景定位第二个数正数往下走，负数往上走。

    如果背景图超出了元素的边框那么多出来的内容将不会显示出来。

利用上面的技术可以实现精灵图（雪碧图、CSS Sprites），是一种网页图片的应用处理方式，将一个页面中很多的零星的小图片包含到一张大图里面去。为了是节省服务器的请求次数。

background-color

background-image

background-position

background-repeat

简写：background:image position repeat color

图标字体

图标字体也是一种字体。

使用步骤：

  1. 定义图标字体

    让浏览器可以从web服务器下载字体，这就以为这不必依赖用户机器中的字体也能让用户看到设定的字体。

     @font-face {//自定义一个字体
   font-family: 'xiaobaobei';//自定义字体的名字
     /*
     每种浏览器可以识别的字体的后缀不一样
     */
   src: url('iconfont.woff2?t=1660114637797') format('woff2'),
        url('iconfont.woff?t=1660114637797') format('woff'),
        url('iconfont.ttf?t=1660114637797') format('truetype');
 }

  2. 使用图标字体

     /*定义一个类选择器他叫iconfont*/
 .iconfont {
     /*我要用什么字体*/
   font-family: "xiaobaobei" !important;
   font-size: 16px;
   font-style: normal;
     /*
     CSS3中的属性，让字体抗锯齿。
     让字体看起来更加清晰，丝滑。
     */
   -webkit-font-smoothing: antialiased;
   -moz-osx-font-smoothing: grayscale;
 }

  3. 挑选相应图标并获取字体编码，应用于页面

     <span class="iconfont">&#x33;</span>

厂商前缀（供应商前缀）

如果在看源代码的时候发现-webkit-类似开头的代码，这种东西叫厂商前缀。浏览器厂商用它们来标识出来实验性的属性或浏览器私有属性。

列表

每个列表默认的时候都会给自己预设一些样式。（内边距（左）、外边距（上下））缩进列表项。

CSS中的每个列表项都会包含一个标记，这个标记不参与文档布局，只是附加在列表的一侧（不管列表项在哪里标记都会随着列表的移动而移动）

列表项会生成两个盒子（标记盒子、内容盒子）

列表类型：list-style-type:value

  • none，没有列表标识。
  • 其他的值自行百度。

列表图像：list-style-image:url()

  • none，默认不设置列表图像。
  • url(图像地址)

列表记号的位置：list-style-position

  • outside，默认值，将标记放在标记盒子中。
  • inside，将标记放在内容盒子中。

列表样式的简写：list-style:type image position

顺序可以更改。

单元格的边框

border-collapse，写到table元素上。

  • separate，分离边框，边框之间有距离。
  • collapse，单元格的边框之间没有间隔，他们之间的边框将合并或折叠在一起。
