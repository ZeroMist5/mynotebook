内联盒子模型

内联盒子模型

块状元素中通常放置的是一行一行的行内元素，内联盒子模型决定了一行一行的行内元素该如何摆放。

<p>这里是普通的文字，这里有一个<em>em</em>标签！</p>

内联盒子的类型：

  • 如果内容被行内元素所包含则属于内联盒子（具名）。
  • 如果只是光秃秃的文字，则属于匿名的内联盒子。

内联盒子组成：一个内联盒子由内容区、上半行、下半行间距组成。

内容区是用来放置具体内容的。

上下半行艰巨是在内容区的上方和下方的一块空白。

行框盒子（line box），每一行就是一个行框盒子，每个行框盒子是由一个一个的内联盒子组成。行框盒子需要将内联盒子包裹住，所以本行的最高内联盒子的上边界决定了行框的上边界，本行的最低的内联盒子的下边界决定了行框的下边界。

行高

格式：line-height，定义行高。

块状元素没有设置高度，默认的时候其高度由行高了决定。

  • normal，默认值。跟用户的浏览器走。

  • px，像素

    我们用的时候是将line-height放在块状元素上。line-height放在块状元素上表示的是每行的最小的行框高度。（行与行之间的最小距离（可以大于等于））。

    既然是每行的最小值，那么说明可以设置比该值大。

  • line-height的值也可以是number（一个数值），将数值作为行高的值。可以根据当前的font-size大小进行计算，表示当前font-size的多少倍。

    20px line-heihgt=1.5 1.5* 20 = 30px

    上面所说就说line-height的第一个功能：调整行与行之间的距离。

    line-height具有继承性。

    不设置块状元素高度的情况下，其line-height改变会撑高盒子高度，并且文字会垂直居中于盒子。并且盒子的高度会和line-height的高度是相同的。(撑开盒子的是行高而不是内容)

  <html>

      <style>
          div{
              border:1px solid green;
              line-height: 71px;

  <body>

      <!DOCTYPE html>
  <html>

  <head>
      <style>
          div{
              border:1px solid green;

  </body>

  </html>

    line-height的第二个功能，单行文本垂直居中的时候将line-height和height的高度设置为相同。

垂直居中

baseline（基线），基线是内联盒子中小写字母的x的下边缘。为了排列整齐用的。

vertical-align:value

  • baseline，默认值。强制元素的基线对齐。

    如果是图片，图片的底端与x的基线对齐。

  • bottom，将内联盒子的底边与行框的底边对齐。

  • top，将内联盒子的顶边与行框的顶边对齐。

  • middle，元素的垂直中心点（中线）和父元素的x的1/2处对齐。

vertical-align中使用的问题

  • 在一行中，一个高的元素占据了整行的高度，vertical-align对它不会有影响（因为没有空间），这个时候还需要有对齐的效果，所以会导致父元素的基线移动。

      <!DOCTYPE html>
  <html>

  <head>
      <style>
          body{
              font-size:50px;
          }
          img{
              /* vertical-align: middle; */
              vertical-align: middle;
          }
      </style>
  </head>

  <body>
      nx

  </body>

  </html>

  • 块状元素只包含一张图片的时底部会有不知名的空白。

      <!DOCTYPE html>
  <html>

  <head>
      <style>
          div{
              border:1px solid green;
          }

  <body>

    问题分析：

    1. 在HTML5文档声明中，行内元素的所有解析会表现为在每个行框盒子的前面有一个空白节点，这个空白的节点透明，不占据宽度，没有办法用脚本获得（幽灵节点）。只会在<!DOCTYPE html>的时候才会出现。

    2. 根据上面所说空白节点在图片前面有一个看不见的内容。既然有内容就会后上下半行间距就会有内容区。默认的时候就是vertical-align:baseline。既然是baseline基线对齐那么从基线的下面一直到这个内联盒子的底部就会有一些空白存在。div边框的空白就是这些导致的。

  </html>

          }
      </style>

          <img src="./10018-1.jpg" alt="">
      </div>
      <span style="font-size:0;">abc</span>
  </body>

    5. 方法三：将图片设置为vertical-align:middle\top\bottom

        <!DOCTYPE html>

 <html>

             height:500px;
             /* 2. 设置line-height和高度相同，这样就能让x移动到垂直中心 */
             line-height: 500px;
             border:1px solid green;
         }
         span{
             width:450px;
             /* 1. 让span变为行内块状元素，这样就能当成一个整体来处理 */

             line-height: normal;
             /* 4. 让span的垂直中心与x的1/2处进行对齐 */
             vertical-align: middle;
             border:1px solid red;
         }

     </style>
 </head>

 <body>

     </div>
 </body>

 </html>

  3. 当使用vertical-align:middle的时候只是近似垂直居中。

    vertical-align:middle是和父元素的x的1/2处对齐。

    字体在设计的时候不会正好占据内容区的1/2处，一般来说都会有一些下沉。

    当字体设置的越大的时候，x就会越大，下沉的就会越多。所以上边和下边的差别也就越大。

    将font-size设置为0，将字体大小设置为之后就会让内容区中的所有线都在中间显示，不存在什么下沉的问题了。

             width:500px;
             height:500px;
             line-height: 500px;
             border:1px solid green;
             font-size:0;
         }
         img{
             vertical-align: middle;
         }
     </style>
 </head>

 <body>
     <div>
         <img src="./2022-08-08_150439.jpg" alt="">
     </div>

盒子模型

文档中的每个元素都会生成一个矩形框我们称为元素框，这个框用来描述元素在文档布局中所占的空间。

  • 内容区：放置具体内容的
    • 宽度：用来定义内容区的宽度。
    • 高度：用来定义内容区的高度。
  • 内边距：用来定义内容区和边框的距离。
  • 边框：边框是内容区或内边距周围的一条或多条线。
  • 外边距：定义元素和元素之间的距离。是在元素周围添加的额外的空白。

宽度

元素的宽度指的从内容区左边界到右边界的距离。

宽度可以设置在块状元素和行内块状元素上。

格式：width:value

  • auto，默认值。
    • 对于块状元素，宽度默认是父元素的宽度。
    • 对于行内元素，宽度默认的时候是内容的宽度。
  • px，像素。
  • %，使用百分比来设置宽度，相对于包含的块状元素的宽度来设置。

高度

元素的高度从内容区的上边界到下边界的距离。

格式：height:value

高度可以设置在块状元素和行内块状元素上。

  • auto，默认值。
    • 块状元素，看行高。
    • 行内元素，即使设置行高也不会影响元素的高度，但是会影响元素的垂直位置。
  • px，像素。
  • %，使用百分比来设置高度，相对于包含的块状元素的高度来设置。

body元素的百分比的宽度和高度依赖于html元素的宽度和高度。html是靠body撑开的、body是靠里面的内容撑开的。

给html设置百分比。

<!DOCTYPE html>
<html>

<head>
    <style>
        /* 需求：让#f占据当前窗口的50%、50% */
        html,body{
            height:100%;
        }
       body{
        margin:0;
       }

       #f{
        width:50%;
        height:50%;
        background-color:lightsalmon;
       }
    </style>
</head>

<body>
    <div id="f">
    </div>
</body>

</html>

最小宽度、高度，最大宽度、高度

  • 元素的内容区可以使用min-width和 min-height定义最小值。取值和widht、height相同。

    最少（可以大于等于）

    买苹果，10块钱一斤，讲价9块。你只能最少给9块，多了不限。

  • 元素的内容区可以使用max-width和max-height定义最大值。取值和width、height相同。

    最多（可以小于等于）

    3000，做多只能是3000，只能少不能多。
