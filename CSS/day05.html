CSS基础

CSS

选择器{
    属性1:值1;
    属性2:值2;
    .....
}

颜色：

  • 关键字
  • 6位的16进制的字符串。
  • rgb()
  • rgba()

css的使用方式：

  1. 行内样式

    使用标签的style属性

  2. 内嵌样式

    使用style标签

  3. 外部链接

    <link href="xxx" rel="stylesheet" type="text/css"/>

    href属性：链接的文件的地址。

    rel属性：relation，关系。链接进来的文件和当前文档的关系。

    type属性：text/css，引入文档的mime类型。

选择器

  • 标签选择器，使用HTML标签的名字作为选择器。

  • *，通配符选择器，选择所有的标签。

  • 类选择器

    • 在CSS中定义的

        .类名{
  }

    • 在HTML中使用class属性，属性的值是类名

      一个类可以有多个名字。

      .a.b，同时具有a这个类和b这个类的将会被选中。

      <!DOCTYPE html>
<html>

<head>
  <style>
      /* 
      你找男朋友  要180+ 有钱  长的帅

      180 有钱 帅  体贴
       */
       /* 选中同时具有a和b的。 */
      .a.b{
          color:pink;
      }
  </style>
</head>

<body>
  <span class="a b c">呵呵</span>
</body>

</html>

  • ID选择器

    ID，唯一的标识。

    步骤：

    1. 在CSS中使用#id的值作为选择器

    2. 在HTML中使用id属性，id属性的值是#号后面的值。

      <!DOCTYPE html>
<html>

<head>
 <style>
     #hehe{
         color:yellowgreen;
     }
 </style>
</head>

<body>
 <b id="hehe">略略略</b>
</body>

</html>

      注意：

    3. id相当于身份证，具有唯一性在HTML文档中一个id的值应该只出现一次。

    4. ID的值是区分大小写的。

  • 组合选择器

    选择器1,选择器2,选择器3........

  • 后代选择器，格式：E F，用来选择E元素中的F元素。F元素是E元素的后代。

    儿子、孙子、重孙子........

      <!DOCTYPE html>
  <html>

  <head>
      <style>
          /*div里面的所有的b标签。b标签可以是div的儿子、孙子、重孙子.......*/
          div b {
              color: red;
          }
      </style>
  </head>

  <body>
      <div>
          <b>呵呵1</b>
          <p>哈哈
              <b>略略略2</b>
          </p>

          <b>哈哈哈3</b>

      </div>
      <b>嘿嘿</b>
  </body>

  </html>

    后代选择器，两个选择器之间需要有空格。

  • 链接伪类选择器

    链接伪类选择器只针对于html中的a标签。只有两种状态。未访问过，已访问过。

    • a:link，选择匹配的a标签，a标签定义了href属性并且没有被访问过。

    • a:visited，选择匹配的a标签，a标签定义了href属性并且已经被访问过。

      访问过之后就存在了浏览器记录，再开打也是访问过。

      不同的浏览器他的历史记录不是共享的是分开的。

  • 动态伪类选择器

    在某种情况下会被触发，并且不单单限于a标签。

    • E:hover，选择匹配的E元素，且用户鼠标停留在E元素上的时候会被触发。

    • E:active，选择匹配的E元素，且匹配的元素被激活的那一瞬间被触发。

      如果在a链接上同时使用那么需要遵循一个顺序link->visited->hover->active

      爱恨原则：LoVe/HAte，link->visited->hover->active。

      没有访问过（link）-》已经访问过（visited）

      先移动到上面（hover）-》触发点击（active）

权重值

同一个元素有可能被多个选择器同时选中，他们又应用了相同的属性规则（在这个时候就产生了属性值的冲突），在这个时候需要计算权重值。

为什么你们要这么关注权重值。我在此立贴为证：你写页面的时候很少会去计算权重值。

权重值由4部分组成：0，0，0，0

  • ID选择器，为0,1,0,0
  • 类、伪类选择器，为0,0,1,0
  • 标签选择器，为0,0,0,1
  • 通配符选择器，0,0,0,0
  • 行内样式，1,0,0,0

比较权重值的时候从左边每一位开始比较，数值越大权重越高。

如果权重值相同时候最后一个样式胜出。

如果说不需要比对权重值的情况下我们可以在属性后面加!important，使用!important时会忽略权重。

!important -> 权重值 -> 浏览器自带的样式 -> 继承下来的样式

!important能不用就不用。少用。

劲酒虽好不要贪杯。

字体

  • 字体

    字体是文字的形式特征。是文字的衣服。

    C:\Windows\Fonts文字存储路径

    字体可以分为免费使用和商业使用。

  • 字体的组成

    我们认为的字体有可能不单纯。有可能我们所说的字体由多个字体的变体组成。这些变体用来描述粗体、斜体、等等。也就是说他有可能是一个系列。

  • 字体族的类型

    • 衬线字体，笔画开始、结束的地方有额外的装饰，粗细有不同的地方。

    • 非衬线字体，没有额外的装饰。笔画的粗细都差不多。

    • 等宽字体

      字母占据的宽度是一样的。针对于英文来说。

    • 手写字体

    • 奇幻字体

  • 设置字体系列

    font-family:字体/字体族的名字

    访问时所使用的字体是客户端的字体。

    我们在设置的时候可以将font-family写成一个字体的列表。定义多种字体。这个时候浏览器根据字体列表检索用户系统中的字体，从左到右。

    font-family: DanZin,华文彩云;

    如果到最后都没有找到这些字体那么使用默认字体。

    我们常用的字体为了规避一些乱码的风险我们通常会使用对应字体的英文名。（具体参见Excel表），如果字体名字包含空格那么尽量使用引号扩起来。

  • 字体大小

    没有用CSS指定文本大小，web浏览器会使用预先设定好的大小。通常是16px，我们也给他叫基准文本尺寸。

    网页中最小的字号是12px。

    font-size：

    • px，像素。

      font-size:0是可以生效的。

    • em，相对于从父元素继承下来的大小来计算。（相对的单位）

      1em=1个继承下来的大小。

    • rem，它的值相对于根元素（html标签）的文本尺寸来进行定义。

  • 字体粗细

    font-weight

    • normal，正常字体。

    • bold，粗体。

    • 100~900，使用100~900作为关键字是因为通常字体或字体族映射了9级的粗细程度。

      主要看使用了哪种字体。字体如果映射了本级的粗细就可以有效果。根据规范来说只要当前设置的粗细程度不比前面的细就符合规范。

  • 字体风格

    font-style

    • normal，设置为正常字体。
    • italic，设置成斜体。

  • 简写方式

    font:style weight size family

    font-style和font-weight的顺序是可以颠倒的。

    font-size和font-family的顺序不能点到并且font中必须声明这两个值。

  • 字体颜色

    color:value使用前面所说的颜色值。

  • 字体下划线

    text-decoration，用来控制下划线。

    • none，标准文本，去掉下划线。
    • underline，下划线。
    • overline，上划线。

    • line-through，贯穿线。

      你可以使用text-decoration一次性给元素加上多种线text-decoration: underline overline line-through;

  • 字符间距

    格式：letter-spacing:value

    • normal，等同于letter-spacing:0

    • px，像素。

      px可以设置为负数，设置为负数之后文字和文字有可能会产生重叠。（除非是故意为之，否则不要这样做）

  • 盒子模型的基本要素

    • width：设置元素的宽度
    • height：设置元素的高度
    • background-color：背景色
    • border，边框 1px solid green，设置1个像素的宽度的边框，这个边框是实线，绿色。

  • 元素的显示模式

    • 块级元素（block）

      • 独占一行，多个块状级元素会自己重启一行。
      • 默认情况下block元素宽度自动填满父元素的宽度。
      • 块状元素可以设置宽度和高度。

      • 设置的元素的宽度和高度可以比父元素的宽度大也可以比父元素的宽度小。

        块状元素：div、h1~h6、form、ul、ol、p。。。

    • 行内元素（inline）

      不会独占一行，多个相邻的行内元素会排列在同一行里，一行放不下就换行。

      行内元素设置宽度和高度无效。

      span、b、string、i、em、u

    • 行内块状元素（inline-block）

      在内部表现为块状元素，可以设置宽度和高度，外部像行内元素水平排列。

      img，表单元素。

  • 更改显示模式

    display:value

    • block，显示为块状元素
    • inline，显示为行内元素
    • inline-block，显示为行内块状元素
    • none，隐藏元素。

  • 文本缩进

    text-indent:value

    • px，像素

    • em，相对于自身的font-size的大小来进行计算。

      注意文本缩进只对第一行产生缩进效果。

      能用在块状元素和行内块状元素上。不能应用于行内元素。

  • 换行

    格式：white-space，用来处理元素内的文本是否允许换行。(列表的时候用，不允许换行)

    • normal，默认值。CJK文本（china中国，japan日本，korea韩国）遇到容器边界自动换行，非CJK文本由word-break决定。

    • nowrap，超出容器边界不换行。

      格式：word-break，用来标明怎样进行单词内的断词。（文章内容的时候用，必须得换行。）

    • normal，根据浏览器机默认设置决定是否进行换行。

    • break-all，软换行会出现在任何字符之间（主要针对英文）

  • 超出之后

    overflow:value

    • visible，超出元素框的内容依然可见。
    • hidden，超出元素框的内容隐藏起来。
    • scroll，超出的内容以滚动条的形式显示出来。
    • auto，浏览器自己确定，具有不确定性。

  • 水平对齐

    text-align:value

    • left，文字放到左边。
    • right，文字放到右边。

    • center，文字放到中间。

      首先：text-align用在谁身上（text-align用在块状元素和行内块状元素上），其次对什么东西生效（对自身不生效，对内部的行内元素和行内块状元素生效）。
