01-JavaScript概述

前端三大件：

  • html: 负责网页的骨架
  • css：负责网页的美观
  • javascript(js)： 负责网页的动态效果和交互行为，从而提高用户的体验

JavaScript是什么

  1. JavaScript是一种运行在客户端浏览器中的脚本语言。
  2. 脚本语言：不需要编译，运行的过程中由js解释器（js引擎）逐行进行解释并执行。

    如 java语言，先要编译，然后再执行。

JavaScript作用

  • JavaScript诞生于1995年，其作者是 布兰登-兰奇，仅仅利用十天完成JavaScript设计，它的出现主要是用于处理网页中的前端验证。

  • 所谓的前端验证，就是检查用户输入的数据是否符合一定的规则，比如用户名长度，邮箱格式等。

  • 现在的主要场景如下：

    • 轮播图
    • 商品放大镜
    • 表单验证。
    • 游戏开发
    • 服务器开发（借助Node.js平台，可以让JavaScript运行在服务端）。
    • .....

JavaScript代码编写的位置

编写位置和css一样，行内式、嵌入式、外链式

先来学习两个基本JS语句：

打开浏览器控制台console:

alert(数据); // 浏览器弹窗显示数据
alert('你好')
alert("你好","世界")
alert(520)

console.log(数据); // 向浏览器控制台打印数据 (便于后面代码调试)
console.log(3+3)
  • 嵌入式
<head>
  <script>
    // 弹窗
    alert('你好!');
    alert('JS');
  </script>
</head>

一个语句用一个分号结尾，当然这个分号是可选的，建议初学者还是加上，养成一个良好的编码习惯

浏览器控制台中打印结果，打印多个值用逗号隔开。

  • 行内式
<input type="button" value="按钮" onclick="alert('Hello World')" style="color：red"/>

控制台打印输出：

<head>
  <script>
       // 控制输出
       console.log('你好!');
       console.log('JS');
  </script>
</head>
  • 外链式
<head>
  <script src="index.js"></script>
</head>

<!-- 错误写法 -->
<script src="index.js">
  alert('你好'); // 不会执行的。引入外部的js文件中，script段中定义的代码是不会执行的
</script>

注意：

  • 建议把css定义在网页头部，及时渲染。

  • 建议js脚本定义在网页body内部后,防止页面阻塞。

    在浏览器中有两个线程：JS引擎线程和UI渲染线程。这两个线程是互斥的

  • 无论代码定义在script段内，还是script引入的文件中，代码都是按照从上到下的顺序执行

JavaScript的组成部分

分为三部分：

ECMAScript - 定义JS的核心语法

为了确保不同的浏览器上运行的JavaScript标准一致，所以几乎所有公司共同定制了JS的标准命名为ECMAScript。

ECMAScript（ECMA：全称是欧洲计算机制造联合会，此机构定义了JavaScript的核心语法规范标准、和具体的实现。

ECMAScript版本演进：

  • JavaScript 的第三版ES3，从 1999 年开始
  • 2009 年发布，ES5，
  • 2015年，ECMAScript 6发布正式版本，简称 ES6
  • 2016年，ECMAScript 7发布正式版本，简称 ES7
  • 2017年，ECMAScript 8发布正式版本，简称 ES8
  • 2018年，ECMAScript 9发布正式版本，简称 ES9
  • ......

现在的我们学的都是ES5的及之前核心语法。 后面课程会陆续讲解ES6/7/8/9的高级新特性。

由于ECMAScript是一套标准，所以不同浏览器厂商实现起来也会有不同的实现。

浏览器 JS引擎（Javascript实现方式）
IE/Edge JSScript(<IE9); Chakra(IE9+及Edge)
Firefox spiderMonkey
Chrome V8（记住）

可见，浏览器本身并不会执行JS代码，而是交给它内部JS引擎去执行。

当然，现代浏览器JS语法基本都是兼容的。 (IE: 你们看我干嘛！夹菜啊)

DOM - 文档对象模型

  • DOM （ Document Object Model）：文档对象模型，主要操作页面节点元素。
  • 如改元素颜色、改字体大小、删除元素、添加元素等。

案例演示：

如：点击一个按钮，让一个div文字颜色被红色，背景变绿色。

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        #box {
            width: 100px;
            height: 100px;
            color: pink;
            background-color: blue;
        }
    </style>

</head>

<body>
    <div id="box">我爱你们</div>
    <button id="btn">变吧</button>
    <script>
        // console.log(document.getElementById('box').innerText)

        // DOM操作，操作页面节点
        document.getElementById('btn').onclick = function(){
            // 让id=box元素的文字颜色变红色，背景变绿色
            var boxEle = document.getElementById('box')
            boxEle.style.color = 'red';
            boxEle.style.backgroundColor = 'green';
        }
    </script>
</body>

</html>

BOM - 浏览器对象模型

BOM（Browser Object Model ）：浏览器对象模型。主要操作浏览器相关功能。

案例演示：

如：点击按钮让浏览器刷新

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>

</head>
<body>
    <button id="btn">刷新</button>
    <script>
        // DOM操作体验
        document.getElementById('btn').onclick = function(){
            // 让浏览器刷新
            location.reload();
        }
    </script>
</body>
</html>
