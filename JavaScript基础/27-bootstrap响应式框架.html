27-BootStrap响应式框架

Bootstrap响应式框架的使用

版本号：基于bootstrap:v3.4.1

网址:https://v3.bootcss.com/

要求

  • HTML5文档

  • 移动设备优先(HTML5文档默认会有，不用单独设置)

    <meta name="viewport" content="width=device-width, initial-scale=1">

Normalize.css

为了增强跨浏览器渲染的一致性，bootstrap内部使用了 Normalize.css CSS 重置样式库

css 重置化（reset）和正常化（normalize）区别:

  • css reset重置化： 重置意味着去除所有的浏览器默认样式，这样更易于保持各浏览器渲染的一致性。你将必须重新定义各种元素的样式（padding、margin、font-size） 
   * {
        padding:0;
        margin:0;
    }
    ol,li {
        list-style:none;
    }

  • css normalize正常化（ normalize.css ）:并没有完全去掉所有的默认样式，而是保留了有用的默认值，同时还纠正了一些常见的bug错误,使浏览器具体通用性

    如IE不支持main标签，而 normalize.css修复了此bugs

布局容器

两个类：

  • .container 类 ：用于固定宽度并支持响应式布局的容器。默认的宽度是1170px ，且在不同的屏幕尺寸下面宽度是不一样的
  • .container-fluid 类 ： 占据视口宽度的 100% ，不是响应式的。

注意，由于 padding 等属性的原因，这两种 容器类不能互相嵌套

.container 类 用于固定宽度并支持响应式布局的容器。默认的宽度是1170px ，两边会留白

<div class="container">
  ...
</div>

<div class="container-fluid">
  ...
</div>

栅格系统（响应式原理）

采用栅格化系统。 根据媒体查询把设备分成多个尺寸（4个），然后把一行分为12列，在不同设备宽度下面会应用不同的响应式类名。

栅格参数：

  • 行（row）”必须包含在 .container 或 .container-fluid 类中，以便为其赋予合适的排列（aligment）和内补（padding）。
  • 通过“行（row）”在水平方向创建一组“列（column）”。
  • 你的内容应当放置于“列（column）”内，并且，只有“列（column）”可以作为行（row）”的直接子元素

媒体查询

在栅格系统中，我们在 Less 文件中使用以下媒体查询（media query）来创建关键的分界点阈值。 

/* 超小屏幕（手机，小于 768px） */
/* 没有任何媒体查询相关的代码，因为这在 Bootstrap 中是默认的（还记得 Bootstrap 是移动设备优先的吗？） */

/* 小屏幕（平板，大于等于 768px） */
@media (min-width: @screen-sm-min) { ... }

/* 中等屏幕（桌面显示器，大于等于 992px） */
@media (min-width: @screen-md-min) { ... }

/* 大屏幕（大桌面显示器，大于等于 1200px） */
@media (min-width: @screen-lg-min) { ... }

示例代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="./bootstrap-3.4.1-dist/css/bootstrap.css">
    <style>

        .row div:first-child {
            background-color: pink;
        }

        .row div:nth-child(2) {
            background-color: #ccc;
        }

        .row div:last-child {
            background-color: skyblue;
        }
    </style>
</head>
<body>
    <div class="container">
        <!-- 
            在不同设备下应用不同的类名
            大屏幕（>1200）：应用类名：col-lg-数字  
            中屏幕（>992）：应用类名：col-md-数字 
            平板屏幕（>768）：应用类名：col-sm-数字
            移动设备（<768）：应用类名：col-xs-12 ,默认占据12列（即占满父容器一行）
        -->
        <div class="row">
            <div class="col-lg-4 col-md-8 col-sm-2">1</div>
            <div class="col-lg-6 col-md-2 col-sm-5">2</div>
            <div class="col-lg-2 col-md-2 col-sm-5">3</div>
        </div>

    </div>
</body>
</html>
