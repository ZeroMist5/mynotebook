12-DOM

DOM操作

回顾js组成部分

三大部分：

  • ECMAScript: 规定js语言的实现标准和规范。

  • BOM（Browser Object Model ）：浏览器对象模型。它提供了一套操作浏览器功能的API方法，可以操作浏览器窗口。比如：控制浏览器前进、后退、刷新、获取屏幕分辨率等

    简单理解：API（Application Programming Interface，应用程序接口）就是程序预留给程序员提供的一种工具，以便能够轻松的实现某些功能。如 弹窗 alert就是一个API。

  • DOM （ Document Object Model）：文档对象模型。它提供了一套操作页面元素的API。作用：便于操作页面节点。如操作标签的颜色，背景，添加元素、删除元素等等。

我们先讲解DOM相关知识。

DOM每个字母的含义：

  • （Document）文档：表示的就是整个HTML网页文档.
  • （Object）对象：表示将网页中的每个节点（如div）都视为了一个节点对象
  • （Model）模型：描述了文档中每个节点之间的关系，这样便于我们操作整个文档。

HTML文档

一个HTML网页就是一个文档，其中的各种节点组成了DOM树。 document就是文档对象

节点

  • 节点（Node）： 是构成HTML文档最基本的单元。
  • 节点分类:
    • 文档节点：整个HTML文档，也可称document节点
    • 元素节点：HTML文档中的某个HTML标签，如a节点,div节点
    • 属性节点：元素的属性，如 a标签的href属性
    • 文本节点：标签中文本内容。如 h1标签中的内容
    • 注释节点：我们所写的注释
// div元素节点、id属性节点、content文本节点
<!-- 我是注释节点 -->
<div id="box">content</div>

document文档对象属性

属性名 功能
document.body 获取文档中的body标签对象（DOM节点、标签对象、元素）
document.documentElement 获取文档中根元素 html标签
document.forms 获取文档中所有表单元素 返回是一个集合
document.images 获取文档中所有的图片元素 返回是一个集合

凡是在JS中遇到“集合”两字 它就是一个伪数组 ，它的访问方式与数组一样，可通过下标来访问。也可以for循环遍历，也可以通过length长度来获取匹配的元素个数

document查找节点的方法

想要找到页面中的某个节点进行操作，必须先找到它，才可以进行下一步的操作。

以下方法都是在文档中找节点。

  • document.getElementById("ID名") : 通过id的属性值来获取节点对象。没找到,则返回null

  • document.getElementsByTagName("标签")： 通过标签名来获取节点对象

  • 祖先节点.getElementsByTagName("标签名") : 也可通过祖先节点来获取其下面的指定标签名节点。

  • document.getElementsByClassName("类名") : 通过类名来获取节点对象

  • document.getElementsByName('name名') ：通过标签的name属性值来获取元素。

  • document. querySelector(selector) ：返回选择器所匹配的第一个节点对象。

  • document. querySelectorAll(selector) ： 返回选择器所匹配的所有元素。

    参数selectors:相当于css选择器

上面的方法除了document.getElementById("ID")和document. querySelector(selector)都是返回一个指定的节点对象，其他方法都是返回一个节点集合（伪数组）。 只要带Elements的都是返回集合。

示例代码：

  1. 通过id获取节点
<body>
    <div id="box">小猪佩奇</div>
    <script>
        // 获取id=box的元素节点
        var boxEle = document.getElementById('box')
        // 设置文字颜色红色
        boxEle.style.color = 'red'
        // 设置内容
        boxEle.innerText = '冷冷的冰雨'
    </script>
</body>
  1. 通过标签名获取节点
<body>
    <ul>
        <li class="item">佩奇</li>
        <li class="item">乔治</li>
        <li class="item">猪爸爸</li>
    </ul>
    <script>
        // 获取标签名为li节点对象
        var lisEle = document.getElementsByTagName('li')
        console.log(lisEle); // HTMLCollection(3)
        // 修改集合中的下标为2的节点对象内容
        lisEle[2].innerText = '猪爷爷'
    </script>
</body>
  1. 也可以通过类名获取：
<body>
    <ul class="list">

        <li class="item">猪爸爸</li>
    </ul>
    <script>
        // 获取类名为item的节点
        var lisEle = document.getElementsByClassName('item')
        console.log(lisEle); // HTMLCollection(3)
        lisEle[1].innerText = '猪妈妈'
    </script>
</body>
  1. 通过name名称获取

    </script>
</body>
  1. 通过选择器获取节点
<body>
    <ul class="list">

        <li>猪爸爸</li>

</body>
  1. 通过选择器来获取节点集合
<body>
    <ul class="list">
        <li>佩奇</li>
        <li>乔治</li>
        <li>猪爸爸</li>
    </ul>

        console.log(lisEle); // NodeList(3)
        // 修改集合中的下标为2的节点对象内容
        lisEle[2].innerText = '猪爷爷'
    </script>
</body>

注意：querySelector(css选择器)和querySelectorAll(css选择器)的区别：

  • querySelector：仅返回匹配到的第一个节点对象
  • querySelectorAll： 返回匹配到的所有节点对象

节点的类型

节点（Node）： 是构成HTML文档最基本的单元。

  • 常用节点分为四类:
    • 元素节点：HTML文档中的某个HTML标签。平时使用最多的
    • 属性节点：元素的属性
    • 文本节点：HTML标签中的文本内容
    • 注释节点：我们所写的注释

通过节点对象的node.nodeType属性可以获取节点类型，创建节点类型如下：

node.nodeType // 1 元素节点  (element Node)
node.nodeType // 2 属性节点  (attribute Node)
node.nodeType // 3 文本节点  (text Node)
node.nodeType // 8 注释节点 （Comment Node）
node.nodeType // 9 文档节点 （document Node）

其中元素节点用的是最多的！！

示例：

<body>
    <div id="box">
        <!-- 我是一个盒子 -->
        我是一个盒子
        <div class="a b">八戒</div>
        <h3>悟空</h3>
    </div>
    <script>
        // 获取id=box的元素节点
        var boxEle = document.getElementById('box');
        console.log(boxEle.childNodes); // 获取所有的子节点
        console.log(boxEle.children);// 获取所有的子元素节点


        console.log(boxEle.childNodes[3].nodeType); // 1 节点类型
        console.log(boxEle.childNodes[3].nodeName); // DIV 节点名称
        console.log(boxEle.childNodes[3].localName); // div 节点名称
        console.log(boxEle.childNodes[3].className); // 'a b' 获取类名

    </script>
</body>
  • childNodes: 获取某个节点下面的所有子节点
  • children: 获取某个节点下面的所有子元素节点

JS事件处理

什么事件

  • 事件就是文档或浏览器窗口中发生的一些特定的交互瞬间。
  • 如：鼠标单击、鼠标悬浮、键盘按下抬起等事件。
  • JavaScript与HTML之间的交互就是通过事件实现的。

事件的绑定方式

事件的绑定常用的两种方式：

  • 行内绑定
  • 动态绑定(js)

绑定事件三要素

  • 确定事件源 （给哪个节点对象绑定）
  • 事件类型： 如：onclick单击事件，onmouseover
  • 事件处理的回调函数
  1. 行内绑定

说明：所谓的行内绑定事件 是将事件写在HTML标签里面

格式：<标签名 事件名='函数名()' />

<button onclick="foo()">按钮</button>
<script>
    function foo(){ 
        alert('ok') 
    }
</script>
  1. js动态绑定

即通过JS获取Dom对象,再给dom对象动态去绑定事件的方式。

格式：

DOM对象.事件名 = 事件处理程序

事件的处理程序一般是一个匿名函数或是函数名。


<button id="btn">按钮</buton>

<script>
    // 给id=btn的元素绑定单击事件
    document.getElementById('btn').onclick = function(){
        console.log('被点击了')
    }
</script>

常见事件

  • 页面事件：
    • onload:页面元素结构和资源加载完毕后执行，一般给window绑定。
    • DOMContentLoaded:仅等待页面结构加载完毕后执行，不需等待完毕资源，所以速度快于onload。
    • onresize: 改变浏览器窗口大小触发
    • onscroll: 滚动条滚动触发
<head>
    <script>
        // onload:会等待页面结构和资源（css,image）部加载完毕之后在执行
        window.onload = function () {
            console.log('页面加载完毕了')
            document.getElementById('btn').onclick = function () {
                alert('你好')
            }
        }


        // 页面元素内容（无需等待css、js、images）全部加载完毕后执行事件
        document.addEventListener('DOMContentLoaded',function(){
          console.log('DOMContentLoaded')
        }) 


    </script>

</head>

注意：css一般写在网页的头部（一起渲染页面），js脚本写在网页的下面（主要防止页面阻塞，出现页面白屏情况）。

  • 焦点（光标）事件：给input 输入框绑定

    • onblur：失去焦点
    • onfocus：获取焦点
    <body>
    <!--  autocomplete：关闭缓存提示 -->
    <!-- placeholder：提示占位符 -->
    <input type="text" id="search" placeholder="输入关键字" autocomplete="off">
    <script>
        var searchEle = document.getElementById('search');

        // 获取焦点事件
        searchEle.onfocus = function(){
            console.log('我获取焦点了')
        }
        // 失去焦点事件(必须先获取焦点)
        searchEle.onblur = function(){
            console.log('我失去焦点')
        }

    </script>
</body>

  • 单击事件

    • onclick：单击
    • ondblclick：双击
    <body>
    <div class="box"></div>
    <script>

        // 通过变量缓存起来，避免查找多次，浪费性能
        var boxEle = document.getElementsByClassName('box')[0];

        // 单击事件
        boxEle.onclick = function(){
            console.log('老铁你好')
        }

        // 双击事件
        boxEle.ondblclick = function(){
            console.log('爱心赞')
        }
    </script>
</body>
  • 鼠标事件：
    • onmouseover:鼠标悬浮时
    • onmouseout:鼠标离开时
    • onmousedown:鼠标按下时
    • onmouseup:鼠标抬起时
    • onmousemove:鼠标移动时

onmousedown、onmouseup、onmousemove一般在鼠标拖拽的使用较多

  • 切换（给input或select绑定）：

    • onchange：给select下拉框绑定。切换选项时触发。 也可以给文本域input （type=file）绑定，一般用于上传文件

      设置选中下拉框默认的索引 select.selectedIndex = 3

    • oninput：监听input框内容改变时触发。

<body>
    <select id="province">
        <option value="">请选择</option>
        <option value="广东省" id="gds">广东省</option>
        <option value="2" selected="selected">河南省</option>
        <option value="3">江西省</option>
    </select>
    <button id="gd">让广东选中</button>
    <br>
    <input type="text" id="user">
    <script>
        // - onchange：给select下拉框绑定。切换选项时触发。 也可以给文本域input （type=file）绑定，一般用于上传文件
        // - oninput：监听input框内容改变时触发。   

        var select = document.getElementById('province');
        select.onchange = function(){
            console.dir(this);
            console.log(this.value)
            console.log(this.selectedIndex)
        }

        document.getElementById('user').oninput = function(){
            console.log('值变化了')
            console.log(this.value)
        }

        document.getElementById('gd').onclick = function(){
            document.getElementById('gds').selected = true;
        }
    </script>
</body>

  • 键盘事件（给input绑定）：

    • onkeydown：键盘按下触发（并不需要抬起，若按着不放则一直触发）

    • onkeypress：键盘按下并抬起触发一些功能键键如：shift,ctrl，alt等无法得到识别。

    • onkeyup：（开发使用最多）任何键盘键按下并抬起时触发（任何键盘键都有效）

  • 表单事件
    • onsubmit：表单提交时
    • reset：表单重置时(清除用户输入的信息)

示例：

<body>
    <!-- action="aaa.php" 提交的url地址 -->
    <form action="aaa.php" onsubmit="return submitCallback()" id="loginForm">
        用户名: <input type="text" name="username" id="username"> <br />
        密码： <input type="password" name="password" id="password"> <br />
        <input type="submit" value="登录">
        <input type="reset" value="重置">
    </form>

    <script>
        // 点击form表单中的type=submit的按钮，会默认向action地址中跳转。这是默认行为
        // 跳转之前，起码要校验用户的信息是否合法才可以跳转，不合法则不跳转

        function submitCallback(){
            var username = document.getElementById('username').value;
            var password = document.getElementById('password').value;
            if(username.trim() === '' || password.trim() === ''){
                alert('用户名或密码不能为空')
                return false; // 终止执行
            }
            // 重置表单
            // document.getElementById('loginForm').reset(); // 没有on
            // return true; 会跳转  
            // return false; 不会跳转  
            return true;
        }
    </script>
</body>

节点的文本操作

通过上面的一些方法找到dom节点之后，就可以对其进行一些基本操作。

如：对元素的增删改查,属性操作，类名操作、文本操作、样式操作等等。

元素节点

  • 非表单元素：如a、h1-h6、div、li、span、button、p 即双标签，
    • innerHTML:操作标签内容，含标签
    • innerText: 操作标签纯文本内容，不含标签
  • 表单元素：如：input、textarea、radio、checkbox、select 即单标签:
    • value: 用户表单中输入的值

双标签设置和获取文本：

// 设置值
node.innerHTML = '新值';
// 获取值
var content = node.innerText

表单标签input设置和获取值：

// 设置值
node.value = '新值';
// 获取值
var content = node.value

简单示例：

<body>
    <div id="box">真好，我都是<b>手洗</b></div>
    <input type="text" value="胡畔" id="username">
    <script>
        var boxEle = document.getElementById('box');
        var userEle = document.getElementById('username');
        // 获取
        // console.log( boxEle.innerText ); // 真好，我都是手洗
        // console.log( boxEle.innerHTML ); // 真好，我都是<b>手洗</b>

        // 设置
        // boxEle.innerText = '真好,<b>机洗</b>也可以'
        boxEle.innerHTML = '真好,<b>机洗</b>也可以'


        // console.log( userEle.value )
        userEle.value = '甘敏锷'
    </script>
</body>

示例代码：

<body>
    <ul>
        <li id="a">烂哥</li>
        <li class="b">超哥</li>
        <li class="b">鑫哥</li>
        <li class="b">坚哥</li>
    </ul>
    <script>
        // 需求1： 单击第一个li标签，把内容改为烂哥超级棒，文字颜色改为红色
        var lange = document.getElementById('a');
        lange.onclick = function(){
            this.innerText = this.innerText + '真的超级棒，已经裂开了'
            this.style.color = 'red'
        }

        // 需求2： 单击类名为b的标签，把他们的内容，依次在内容后面加个牛逼两字
        var eles = document.getElementsByClassName('b');
        for(var i = 0; i < eles.length; i++){
            // 绑定事件
            eles[i].onclick = function(){
                console.log(this); // // 获取当前点击的对象
                // console.log(this.innerText ); // 获取值
                this.innerText = this.innerText + '牛逼'
            }
        }
    </script>
</body>

this指向：指向当前调用的dom节点对象。

文本节点

相关属性：textContent

<body>
    <ul id="list">
        你好
        <li>刘亦菲</li>
        <li>郭富城</li>
    </ul>
    <script>
        var list = document.getElementById('list');
        console.log(list.childNodes); // 获取子节点
        console.log(list.children); // 获取子元素节点

        console.log(list.childNodes[0].nodeType); // 3 节点类型（元素节点1， 文本节点3）
        console.log(list.childNodes[0].nodeName); // '#text' 文本节点名称
        list.childNodes[0].textContent = '大家好才是真的好！'
    </script>
</body>

节点的样式操作

行内样式

操作元素的行内style样式，可通过dom对象的style属性进行操作： 如：

DOM对象.style.fontSize = '50px'; //# 设置字体大小为50px
DOM对象.style.backgroundColor = 'green'; // 设置背景色为绿色，值设为16进制如#00ff00也行

// 让元素显示或隐藏
DOM对象.style.display = 'none' ; // 隐藏

        <span>张先锐</span>
    </div>
    <script>
        var spans = document.getElementsByTagName('span');
        // var是全局变量
        for (var i = 0; i < spans.length; i++) {
            (function (i) {
                // 每个函数都会产生一个局部变量i
                spans[i].onclick = function () {
                    // 当前函数作用域找不到i,则找父级作用域i
                    console.log(i); // 0 1 2
                }
            })(i)

        }

        // 为什么单击事件中的i都是3？ 
        // 因为for循环完后，i已经变为3了，循环完毕后再去单击，所以i获取到全局变量最后的值i是3
        // 主线程执行完毕for三次后，浏览器的DOM事件线程在去执行事件，当然i一定是最后的值，即3
        // 解决办法：事件外面套一层函数，形参一个私有作用域，再把变量i传递进去。这就保存成当前的变量i的值，
        // 后面事件处理程序中再获取i就是获取父函数中局部变量i,就不会找到全局变量i了
    </script>
</body>

两个时间器函数

两个时间器函数,也有回调函数。

var timer1 = setTimeout(callback,time): 延时器  （执行一次）
var timer2 = setInterval(callback,time): 定时器  （执行多次）
// 上面函数返回的timer是一个随机非0的数值 

clearTimeout(timer1); // 取消延时器
clearInterval(timer2); // 取消定时器

时间是毫秒为单位。 1s = 1000ms

    // 2s后执行callback回调
    setTimeout(function(){
       console.log('你好')
    },2000)


    // 每隔1s执行callback回调
    setInterval(function(){
       console.log('中国强大')
    },1000)

<body>
    <button id="start">开始</button>
    <button id="stop">停止</button>
    <script>
        var startEle = document.getElementById('start')
        var stopEle = document.getElementById('stop')
        var timer;

        // 点击开始按钮，每隔1s输出:点名中
        startEle.onclick = function(){
            // 如果定时器正在运行，则不在执行定时器
            if(timer){
                // 不往下面走
                return;
            }
            timer = setInterval(function(){
                console.log('点名中')
            },1000)

        }

        // 点击停止按钮，则停止点名操作
        stopEle.onclick = function(){
            // 清除上面的定时器
            clearInterval(timer)
        }
    </script>
