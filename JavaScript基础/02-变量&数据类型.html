02-变量&数据类型

JavaScript变量

变量

什么是变量（生活中）

在生活中，有些东西是固不变的，有些东西则会发生变化。 例如，人的姓名和生日是固定不变的，但心情随时会变，今天开心，明天伤心。

什么是变量（variable）（程序中）：

本质：程序中的变量就相当于在内存中申请了一块空间，可以存储变化的数据。

类似于酒店房间，一个房间可以看做是一个变量。

所以变量存在的作用就是为了方便存储一些数据及修改。。

要知道变量是存储在内存中！，不是硬盘中。两者是有区别的：

  • 硬盘： 持久性存储数据，关机后在打开数据还在。
  • 内存：临时性存储数据，关机后就没有了。

定义变量

先声明，后赋值

  • var 用来声明变量
// 声明变量 age
var age;
// 把13赋值给age
age = 13;

var name;
name = '张三';

console.log(age)

变量在内存中的结构：

  • 同时声明多个变量,变量名中间用逗号隔开。
var age, name;
age = 10;
name = '张三';
  • 同时声明多个变量并赋值
var  name = '张三', age = 13;

变量的命名规范

  • 变量名由字母、数字、下划线、$符号组成，且不能以数字开头。若不满足则程序会报错
  • 不能是JS关键字或保留字，例如：var、if、for、while。因为这些关键字在js中是有特殊含义的
  • 变量名严格区分大小写。age和AGE是两个完全不同的变量。
  • 变量名命名必须有意义
  • 命名方法：
    • 小驼峰法。首字母小写，后面单词的首字母需要大写。例如：userName、userPassword。
    • 下划线法。 单词由下划线进行连接。如user_name
var 1abc // 错
var a1bc // 可以
var _abc // 可以
var -abc // 错
var a-abc // 错

注释

注释就是对代码的解释和说明，其目的是让人们能够明白这段代码是什么意思，从而提高代码的可读性。

注意：注释仅仅是给人看的，不会被计算机执行。

注释分为单行和多行注释

单行注释

// 这是一个变量,用来存储年龄

多行注释

/*
    var age = 18;
    var name = 'zs';
    console.log(name, age);
*/

数据类型分类

为什么有数据类型？

在计算机中，不同的数据所占用的内存空间大小是不同的，为了便于将数据分成所需要的不同的内存大小，于是就有了不同的数据类型。

好比人的姓名："张三"，年龄：18，这些数据的类型是不一样的。

数据类型为两大类：

  • 基本数据类型（primitive data type），也叫原始数据类型。
  • 引用数据类型 (object data type)。

基本数据类型

共6种：

  • number: 表示值为数值
  • string: 表示值为字符串
  • boolean: 表示值为布尔值
  • null: 表示值为空
  • undefined: 表示值未定义
  • symbol: 表示值为符号。es6新增
var age =  18
console.log( typeof users ); // "number"

var name = '小明'
console.log( typeof users ); // "string"

var isFinish = false
console.log( typeof isFinish ); // "boolean"

var un = undefined
console.log( typeof isFinish ); // "undefined"

var n = null
console.log( typeof isFinish ); // "object"

引用类型

特点：可以存储多个值

Object(对象)、Array(数组)进行typeof检测都是返回object类型。

// 对象
var userInfo = {name:"小兰",age:18};

// 数组
var users = ['小罗','小李','小虎'];

console.log(typeof userInfo); // 'object'
console.log(typeof users); // 'object'

对一个变量可以typeof操作符获取变量的类型，会返回下列字符串之一：

  • "number": 表示值为数值
  • "string": 表示值为字符串
  • "boolean": 表示值为布尔值
  • "undefined": 表示值未定义
  • "null": 表示空类型。
  • "object": 表示对象类型。
  • "symbol": 表示值为符号。es6新增
var name = '王大锤'
var age = 18
console.log( typeof name ） // "string"
console.log( typeof age ); // "number"

JavaScript的一大特点，它属于一种弱类型或者说动态语言。 这意味着，程序在运行中，变量类型会随时变化，是不确定！。

var age = 18; // number
age = '你好'; // string

Java就是一种强类型的语言，类型一旦定义就不能在改变了。

number类型

示例：

var num1 = 12; // 数字
var num2 = 3.14 // 小数（浮点数）
var num3 = 0.9; // .9也行， 但不推荐

console.log(typeof num1); // "number"


// 若数值本身就是整数，只是小数点后面跟的是0，那么就会转为整数
var m1 = 2.;
var m2 = 10.0;
console.log(m1,m2); //2 10

注意：在JS中不管是整数还是小数其数据类型都是number类型。

浮点数的精度问题 :

于JS采用 IEEE二进制浮点数算术标准 ( IEEE754 ), 导致浮点数计算精度有一定的误差。

为什么这样子？JavaScript 浮点数陷阱及解法

var result = 0.1 + 0.2;    // 结果不是 0.3，而是：0.30000000000000004
console.log(0.07 * 100); // 7.000000000000001
console.log(0.07 * 1000); // 70
console.log(0.5-0.1); //0.4  正常

浮点数值的最高精度是小数点后面 17 位，但在进行算术计算时其精确度远远不如整数。

所以后面尽量避免使用小数做一些运算。

解决浮点数精度丢失问题：

  1. 如果要对小数进行运算可以先转为整数，最后把结果转换回来即可，这样就不会有精度丢失。
var num3 = 0.1;
var num4 = 0.2;
// 转化为整形，最后把结果在除   回来即可
var sum2 = (num3*10 + num4*10)/10;
console.log(sum2); // 0.3
  1. 使用toFixed()函数指定保留小数点后面几位来确保精度。
console.log((0.1+0.2).toFixed(1)); // 0.3

数值的范围

  • 最小值：Number.MIN_VALUE，这个值为： 5e-324
  • 最大值：Number.MAX_VALUE，这个值为： 1.7976931348623157e+308

5e-324: 科学计数法。代表 5乘以10负324次方

由于内存的限制，JS 并不能保存世界上所有的数值 ,若计算到数值超出JS可以表示的范围，那么会自动转为 Infinity(无穷大)，不可在进一步计算

  • 正无穷大：Infinity
  • 负无穷小：-Infinity
// 如果分子是非0值，分母有符号0 或无符号0，则会返回Infinity或-Infinity
console.log(5/0); // Infinity
console.log(5/-0); // -Infinity

NaN

在js中有一个特殊的值叫NaN（Not a Number）,意思是 不是一个数值，用于表示本来要返回一个数值，但操作失败了，就会返回NaN。

示例：

console.log(0/0); // NaN
console.log(-0/+0); // NaN
console.log('hello' * 5); // NaN

// NaN 不等于包括 NaN在内的任何值
console.log(NaN == NaN); // false

  • 那么如何判断一个数值是否是NaN呢？

  • JS中提供了 isNaN( ) 函数，可以判断任何数据类型是否"不是数值"。

console.log(isNaN(NaN));  //true
console.log(isNaN(10));   //false  10是数值
console.log(isNaN("10")); //false  可以转成数值10
console.log(isNaN("blue"));//true  纯字符串不可以转成数值
console.log(isNaN(true));  //false  可以转换为数值1

字符串的连接

拼接字符串法：挖坑法


var age = 29;
var username = '李现'
// +:两边任意一边只要是字符串，则默认是拼接操作
var info = '名字:'+ username +'，今年'+ age +'岁';
var info2 = "名字:"+ username +',年龄：'+ age;
console.log(info2);

  • 自动转换(隐式转换)

    当运算符在运算时，如果两边数据类型不统一，CPU就无法计算, JS引擎会自动将运算符两边的数据做一个数据类型转换，转成一样的数据类型再计算,这种无需程序员手动转换，而由JS引擎自动转换的方式就称为隐式转换

  • 隐式转换规则：加法字串优先，其次数值类型，其他乘 除 减 都统一转数值类型。

字符串拼接使用加号 +连接

// 做加+法计算时，首先转成统一类型，字串优先，其次都转成数值类型
console.log(10+20);     // 30
console.log("10"+20);   // 1020  等价于 "10"+String(20)
console.log("10"+true); // 10true  等价于 "10"+String(true)
console.log("10"+null); // 10null  等价于 "10"+String(null)

// 没有字串类型，则都转成Number类型
console.log(true+20);  // 21  等价于 Number(true)+20
console.log(null+20);  // 20  等价于 Number(null)+20
console.log(undefined+20);  // NaN  等价于 Number(undefined)+20
console.log(null+true);  // 1  等价于 Number(null)+Number(true)


// 若是做 减-  称* 除/时， 则都统一转成数值类型Number再进行计算console.log("10"*20);    // 200console.log("10b"*20);   // NaN    因为 Number("10b")返回 NaN
console.log("10"-true);  // 9
console.log("20"/"4");   // 5

简单小结三点：

  1. +两边只要有一个是字符串，那么+就是字符串拼接功能
  2. 两边如果都是数字，那么就是算术功能。
  3. true或false和数值型进行运算，会把true转换成1，false转成0

undefined类型

undefined表示为一个未定义的值。

  • Undefined类型只有一个值，就是特殊值 undefined。

  • 当使用var声明一个变量但没有初始化时，JS引擎会自动给变量赋予 undefined 值。

// 声明变量但没有初始化时，默认会赋予 undefined
var message;
console.log(message); // undefined

// 比较变量和 undefined 字面量值时是相等的
console.log(message == undefined);  // true

注意：一般来说，永远不用显式地给某个变量设置 undefined 值，此字面值主要用于比较 

// 声明变量，变显式地赋予 undefined，这种方式没有必要
var message = undefined;
console.log(message == undefined);  // true

以下两种情况会返回undefined类型。

  • 变量仅声明不赋值则值默认是undefined，此变量类型就是undefined类型
  • 对一个未声明的变量进行typeof也会返回undefined类型
var myage;
console.log( typeof myage ); // "undefined"
console.log( typeof myemail ); // "undefined"

总结：一般声明变量的同时都会对其进行初始化，因此用 typeof可以判断变量是否未声明。

Null类型

  • Null类型同样只有一个值，即特殊值 null。

  • 逻辑上讲 null 值表示一个空对象指针(引用)，所以用 typeof 判断一个null时会返回 "object" 的原因

var stu = null;
console.log(typeof stu);  //object

// 变量user是存储一个对象,初始化为null,说明将来此变量是要存储一个对象，
// 这里的null是作为一个对象的占位符存在的。更加有语义化（有意义，有含义）
var car = null; // 对象的占位符
console.log( typeof user ); // 'object'
car = {name:'宝马'}

console.log( typeof user ); // 'object'
// 是否内存空间
car = null;

建议：以后定义一个对象变量时，建议使用 null 来初始化，不要使用其他值，这样，只要检查这个变量是不是null就可以判断对象变量是否被赋予对象的引用。

  • undefined 值是由 null 值派生而来的，因此JS将它们定义为表面上相等。
console.log(undefined == null);  // true
console.log(undefined === null);  // false 类型不等

作用：

  1. 如果将来某个变量要存储一个对象，我们可以将变量初始化为null,

  2. 如果一个对象变量使用过后不在使用，我们可以赋值为null,释放其占用的内存空间

小技巧: 不同数据类型控制台打印的颜色是不一样的。字符串的颜色是黑色的，数值类型是蓝色的，布尔类型也是蓝色的，undefined和null是灰色的

## 转换成布尔类型

#### 真值和假值

假值（**falsy**）：js中假值有6个。false、undefined、null、 0、NaN、""（空字符)。转换成布尔类型都是false
真值（**truthy**）：除了以上6个都是真值。转换成布尔类型都是true

Boolean(变量)  ：可以实现将其它的类型转换为布尔类型

```javascript
var v1 = 1; // true
// var v1 = 0;  // false
// var v1 = 2;  // true  将number类型转换为boolean类型 除了0与NaN以外  其它的都是true
// var v1 = -1;
// var v1 = NaN;  //NaN(not a number) false
// var v1 = null; // false
//var v1 = '';  除了空字符串以外  其它的字符串都是true 
console.log(Boolean(v1));

对于 0 、 ''(空字符串)、 null、 undefined 、NaN 会转换成false， 其它都会转换成true

  • 双感叹号技巧。 !!变量名 转化为boolean类型
var x = 100;
console.log(!!x); // true  等价于Boolean(x)
