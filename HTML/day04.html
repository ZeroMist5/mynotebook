HTML表单标签

复习

  • <form>里面的东西是表单的内容。

  • input标签：

    type属性的值来进行设置其他的属性

    • text，文本框。如果说你的type的值不能够被识别出来默认的时候会被解释成为text。

      • name属性：数据提交到服务器上的标识。

      • value属性：默认值。

      • maxlength属性：最大的输入字符数。

      • disabled，禁用。

      • readonly，只读。

        区别：disabled的数据在提交表单的时候不会被提交，readonly的数据在提交表单的时候依然会被提交。但是他们都不能修改。

    • password，表示一个密码框。

      • name属性：数据提交到服务器上的标识。

      • value属性：默认值。（不常用）

        ERP 资源管理计划。

    • radio，单选框，只选择一个。

      • name属性：
        • 起到分组的作用。
        • 传递到服务器上的标识。
      • value属性：默认的时候如果没有value默认的值都是on。所以我需要给每个选项一个值。
      • checked属性：默认选中。checked="checked" 我们经常写为checked

    • checkbox，复选框，可以选择多个。

      • name属性：传递到服务器上的标识。
      • value：传递到服务器上的值。
      • checked属性：默认选中。可以有多个。

    • file：上传框。

      • name属性：上传到服务器的标识。
      • multiple属性：可以选择多个。

    • submit提交按钮

      • value属性：表示按钮显示的值。

    • image图像按钮，也用来进行提交。

      • src属性：指定要当作提交按钮的图片的位置。

        当你点击提交之后会有一个x和y，x和y表示的是你点击图片的时候坐标是什么。

      • width属性：宽度。

      • height属性：高度。

        耦合，解耦。。。。

    • reset重置按钮。很少用。

label标签，用来为表单定义标注。

for属性：用来和input标签进行绑定，值需要和input标签的id的值相同。

input标签需要有id属性，label标签需要有for属性，在这里input的id属性的值需要和label的for属性的值相同。

label的位置和input的位置可以不用紧挨着。只要for属性和id属性产生了绑定就可以。

  1. 练的不够。
  2. 不让你用快捷方式。

汉字都是等宽的，全局空格就是中文的空格。所以一个空格就是一个汉字的距离。

下拉列表

  • <select>标签：定义一个下拉列表

    • name属性：下拉列表叫什么名字
    • disabled属性：禁用整个下拉列表。

  • <option>标签：用来定义下拉列表中的每一项。

    • value属性：传递给服务器的值。如果不写默认是<option></option>中所标记出来的内容。
    • disbaled属性：禁用本选项。
    • selected属性：默认选中。selected="selected" 简单写法 selected。

  • optgroup，对option进行分组。

    • label属性：用来对分组进行描述。
    • disbaled属性：用来进行整个optgroup分组。

  • 多选（select标签上）

    • multiple属性：可以选择多个。

    • size属性：允许显示几个值。

        <form action="" method="get">
      <select name="car" multiple size="2">
              <option>奥迪-A3</option>
              <option>奥迪-A4</option>
              <option>奥迪-A6</option>
              <option>红旗-H5</option>
              <option>福特-猛禽</option>
              <option>长城-风骏7EV</option>
              <option>东风-猛士M50</option>
      </select>
      <button type="submit">提交</button>
  </form>

  • <textarea>，用来输入大段的文字。

    input type="text"的时候不能进行换行，如果换行将会导致表单提交。

    当你需要内容换行或是大段内容的时候需要使用textarea。

    在textarea标签中的内容会作为默认值存在。

      <textarea>这里是默认值</textarea>

    input type="text"适合输入小段文字。

    textarea适合用来输入大段文字。

  • <button>标签：用来表示一个按钮。

    <button type=""></button>

    type值：button（普通按钮没有任何效果）、submit（提交按钮）、reset（重置）

    区别：

    1. button标签是双标签（双标签里面可以添加其他的东西），但是input标签是单标签。

    2. button比input更加语义化，因为button的意思就是按钮。input的意思是输入。

      button type="button"时点击没有任何效果。更加容易改造。

  • span，无意义的行内元素标签（多个排列成一行）。

CSS

CSS的时候不特殊说明都是重点。

CSS（层叠样式表，cascading style sheet），主要是控制网页的布局和样式。使HTML和样式进行分离。

基本语法：

本质就是选择器来选择、改变他的一系列属性。

选择器{
    属性1:值1;
    属性2:值2;
}
选择器{属性1:值1;属性2:值2;......}

选择器：告诉浏览器应该选中哪个元素。

声明：告诉浏览器如何渲染选中的元素，两部分组成

  • 属性：属性是由CSS预先定义好的，属性名通常由一个或多个单词组成。
  • 值：就是要让某个属性解释为某个值。参数。

注意：每个属性/值一对数据后面要加上分号。一对数据不要横着写。

  • 选择器：HTML标签选择器，用HTML的标签名作为选择器来使用。
  • font-size：设置字体大小，单位px。
  • color：设置字体颜色。

CSS中颜色的表示方法

  • 颜色的单词。

  • 十六进制的数字（0~9a~f），总共6位。#。总共6位分成3组，每两位一组（红、绿、蓝）

    蓝颜知己 ====== 蓝 + 黄====绿。数值越大颜色越浅，数值越小颜色越深。

    总共分成3组，每一组的两个值如果都相同，那么可以简写为3个值。

    #ff4400就可以简写为#f40。

    #ff4200不能简写。必须保证每一组的两个值都相同才能简写。

  • rgb颜色值，格式：rgb(红,绿,蓝)，值是从0~255。

  • rgba，格式：rgba(红,绿,蓝,透明度)颜色取值和rgb颜色相同，但是多了一个透明度，透明度是从0~1之间的值。0为完全透明，1为完全不透明。你可以使用小数.8表示的就是0.8

CSS的使用方式

  • 行内样式：在HTML标签中使用style属性来设置CSS样式。

    格式：<标签名 style="属性1:值1;属性2:值2;"></标签名>

      <h1 style="color:red;font-size:100px;">闫海静真英俊！</h1>
  <h1>陶哥是个好人！</h1>

    特点：所写的样式仅作用于本标签。

  • 内嵌方式：在head标签中使用style标签。

    格式：

      <style type="text/css">
      选择器{
          属性1:值1;
          属性2:值2;
          ......
          属性n:值n;
      }
  </style>

    特点：作用于整个页面

    注意：

    1. style标签的type属性可写可不写。

       type="text/css"

      MIME类型。大类型/小类型

    2. 声明的最后一组属性/值的时候后面可以不加分好，但是强烈建议你加上。

  • 外部链接<link />标签。

    格式：<link href="文件名.css" rel="stylesheet" type="text/css"/>

    • href属性：链接的文档来自于哪里。
      • 链接进来的文档的扩展名应该是.css的。
      • 这个.css文件中不要写其他的代码直接写css代码。
    • rel属性：relation，当前链接的文件与当前的文件的关系。

    • type属性：规定被链接的文档的MIME类型。（可选的）

      特点：只要引入了这个css，可以让其在多个页面中生效。

以上三种方式，当样式冲突的时候，采用就近原则。那个属性离被修饰的内容近进使用哪个。

CSS的特性

  • CSS继承性：被包含的元素拥有外层元素的样式效果。(布局相关的东西不会继承)
  • CSS的层叠性：CSS样式允许重复声明，可以覆盖。

CSS的注释符

/*
要注释掉的内容
*/

快捷键：ctrl+/

选择器

  • 标签选择器，直接使用HTML的标签名。恢复默认的时候用的比较多。

  • *，通用选择器、通配符选择器，选择文档中所有的HTML标签。

  • 类选择器

    类：具有相同特征的事物的总称。

    使用：

    1. 在css中使用.后面跟类名作为选择器

    2. 在html标签中使用class属性，class属性的值是定义的类的类名。

      写页面（需求）之前先把HTML标签写明白喽，用合适的标签表达合适的意思。

      注意：

    3. class的值是区分大小写的。

    4. 可以使用标签名.类名的方式选择只具有指定类名的标签。

       <!DOCTYPE html>
 <html>

 <head>
     <style>
         span.hehe{
             color:yellow;
         }
     </style>
 </head>

 <body>
     <span class="hehe">呵呵span</span>
     <b class="hehe">heheb</b>
     <span>hehe span2</span>
     <b>hehe b2</b>
     <span class="hehe">呵呵呵呵</span>
 </body>

 </html>

    5. 一个标签可以使用多个类。多个类之间使用空格隔开。

       <!DOCTYPE html>
 <html>

 <head>
     <style>
         .segui{
             color:green;
         }
         .jiugui{
             font-size:150px;
         }

     </style>
 </head>

 <body>
     <!-- 声明：本例子只是为了课堂效果，并不代表现实中真的这样。 -->
     <span class="segui">陶书见</span>
     <span class="jiugui">李沛华</span>
     <span class="segui">杨博超</span>
     <span class="jiugui segui">杨洪波</span>
 </body>

 </html>

    6. 类选择器可以将两个或两个以上的类进行合并。

       .jiugui.segui{
     background-color:yellow;
 }

      上面代码的意思就是选择一个标签class的值同时具有jiugui、segui的会被选中。
